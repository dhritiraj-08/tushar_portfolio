# Images Directory

## Add Your Images Here

### Logo
- Place your logo file here (e.g., `logo.png`, `logo.svg`)
- Recommended size: 200x50px or similar aspect ratio
- Supported formats: PNG, SVG, JPG

### Hero Background
- Add your hero background image as `hero-bg.jpg`
- Recommended size: 1920x1080px or higher
- Format: JPG or PNG

### Project Images
- Add project thumbnails and images
- Recommended size: 800x600px or higher
- Name them descriptively (e.g., `project-1.jpg`, `commercial-video.jpg`)

### Profile Image
- Add your profile photo
- Recommended size: 600x600px
- Format: JPG or PNG

## Current Structure:
```
static/images/
├── logo.png          (Your logo)
├── hero-bg.jpg       (Hero section background)
├── profile.jpg       (About section photo)
└── projects/         (Project images folder)
    ├── project-1.jpg
    ├── project-2.jpg
    └── ...
```
