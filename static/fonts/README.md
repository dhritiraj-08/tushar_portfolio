# Fonts Directory

## Add Coolvetica Font Files Here

Place your Coolvetica font files in this directory:

- `coolvetica.ttf` (TrueType Font)
- `coolvetica.woff` (Web Open Font Format)
- `coolvetica.woff2` (Web Open Font Format 2 - recommended)

### Where to get Coolvetica:
1. Download from: https://www.dafont.com/coolvetica.font
2. Or search for "Coolvetica font free download"

### After downloading:
1. Extract the font files
2. Copy the `.ttf`, `.woff`, and `.woff2` files to this directory
3. The CSS is already configured to use these fonts

### Font License:
Make sure to check the font license for commercial use.
